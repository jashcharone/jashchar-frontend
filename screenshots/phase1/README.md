# Phase 1 Screenshots

Capture screenshots in Chrome/Edge devtools.

## Required
For each implemented page, capture:
- Desktop (e.g., 1440px width)
- Mobile (e.g., 390px width)

Name files like:
- `home-desktop.png`, `home-mobile.png`
- `about-us-desktop.png`, `about-us-mobile.png`
- `online-course-desktop.png`, `online-course-mobile.png`

Pages list is in `PHASE1_ROUTES_MAP.md`.

Note: automated screenshot capture is not included in this phase because it requires a dedicated E2E runner + stable CI environment. If you want, Phase 1.5 can add Playwright to generate screenshots automatically.
